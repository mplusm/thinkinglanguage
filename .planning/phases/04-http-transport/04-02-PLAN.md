# Plan 04-02: HTTP Server Transport

## Goal
Add `serve_http(port)` to tl-mcp server so TL MCP servers can accept HTTP connections.

## Tasks

### Task 1: Add axum dependency
File: `crates/tl-mcp/Cargo.toml`

Add `axum = "0.8"` (already used in tl-registry, no version conflict).

### Task 2: Implement `serve_http()` in server module
File: `crates/tl-mcp/src/server.rs`

Add:
```rust
pub fn serve_http(handler: TlServerHandler, port: u16) -> Result<(), McpError>
pub fn serve_http_with_runtime(handler: TlServerHandler, port: u16, runtime: Arc<Runtime>) -> Result<(), McpError>
```
- Create `StreamableHttpService::new(factory, LocalSessionManager, config)`
- Factory clones TlServerHandler (impl Clone on TlServerHandler + ToolDef)
- Mount on axum Router at `/mcp`
- Bind TcpListener to `0.0.0.0:{port}`
- Block until shutdown

### Task 3: Integration test
File: `crates/tl-mcp/tests/integration.rs`

Add HTTP tests: start server on random port, connect with `connect_http`, run list_tools/call_tool/ping.

## Verification
- `cargo build -p tl-mcp`
- `cargo test -p tl-mcp`

None (parallel with 04-01).
