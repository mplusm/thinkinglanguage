# Plan 05-01: AST & Parser for mcp_servers

## Goal
Add `mcp_servers` field to agent AST node and parser.

## Tasks

### Task 1: Add mcp_servers to AST
File: `crates/tl-ast/src/lib.rs`

Add to `StmtKind::Agent`:
```rust
mcp_servers: Vec<Expr>,  // Expressions resolving to McpClient values at runtime
```

### Task 2: Parse mcp_servers in agent blocks
File: `crates/tl-parser/src/lib.rs`

In `parse_agent`, add branch:
```rust
Token::Ident(s) if s == "mcp_servers" => {
    self.advance();
    self.expect(&Token::Colon)?;
    self.expect(&Token::LBracket)?;
    while !self.check(&Token::RBracket) && !self.is_at_end() {
        mcp_servers.push(self.parse_expr()?);
        self.match_token(&Token::Comma);
    }
    self.expect(&Token::RBracket)?;
}
```

### Task 3: Update compiler compile_agent
File: `crates/tl-compiler/src/compiler.rs`

Compile mcp_servers expressions to registers. Use AgentExec instruction's unused d-field to pass the mcp_servers list register.
