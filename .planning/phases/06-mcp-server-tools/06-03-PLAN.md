# Plan 06-03: MCP Server Integration Tests

## Goal
End-to-end tests: TL script acts as MCP server, external client connects and calls tools.

## Tasks

### Task 1: TL server test script
File: `tests/mcp_server.tl`

```
fn echo(message) { return message }
fn add(a, b) { return a + b }

mcp_serve([
    map_from("name", "echo", "description", "Echo back", "handler", echo),
    map_from("name", "add", "description", "Add numbers", "handler", add)
])
```

### Task 2: Integration test
File: `crates/tl-mcp/tests/integration.rs` or Rust test

Spawn `tl run tests/mcp_server.tl` as subprocess, connect as McpClient, call tools, verify results.

### Task 3: Error handling tests
- Unknown tool → error result
- Function throws → error propagated to client
- Client disconnect → server shuts down cleanly
