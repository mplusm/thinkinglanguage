  <action>
Create `crates/tl-mcp/src/client.rs` with:

1. **McpClient struct** holding:
   - `runtime: Arc<tokio::runtime::Runtime>` — shared tokio runtime
   - `service: RunningService<RoleClient, TlClientHandler>` — the rmcp service
   - `server_info: Option<ServerInfo>` — cached server info from handshake

2. **TlClientHandler struct** implementing `rmcp::handler::ClientHandler`:
   - Minimal implementation — return defaults/errors for sampling, elicitation, roots
   - `get_info()` returns ClientInfo with name="tl", version from env!("CARGO_PKG_VERSION")
   - Capabilities: empty for now (no sampling, no roots, no elicitation)

3. **McpClient::connect()** static method:
   ```rust
   pub fn connect(
       command: &str,
       args: &[String],
       security_policy: Option<&SecurityPolicy>,
   ) -> Result<Self, McpError>
   ```
   - Check SecurityPolicy: if policy exists and sandbox_mode, call `policy.check_command(command)` — return error if denied
   - Create tokio runtime (or accept Arc<Runtime> parameter for sharing with VM)
   - Use `rt.block_on()` to:
     a. Create `TokioChildProcess::new(tokio::process::Command::new(command).args(args))`
     b. Call `TlClientHandler::default().serve(transport).await` — this performs the 3-step handshake automatically
     c. Cache `service.peer().peer_info()` as server_info
   - Return `McpClient { runtime, service, server_info }`

4. **McpError enum** in client.rs (or a shared error module):
   ```rust
   #[derive(Debug, thiserror::Error)]
   pub enum McpError {
       #[error("Permission denied: {0}")]
       PermissionDenied(String),
       #[error("Connection failed: {0}")]
       ConnectionFailed(String),
       #[error("Protocol error: {0}")]
       ProtocolError(String),
       #[error("Tool error: {0}")]
       ToolError(String),
       #[error("Transport closed")]
       TransportClosed,
       #[error("Timeout")]
       Timeout,
   }
   ```

5. **Drop implementation** for McpClient:
   - Call `rt.block_on(self.service.cancel())` to send shutdown
   - This triggers rmcp's graceful close which closes the transport
   - The TokioChildProcess Drop impl will kill the child process

IMPORTANT:
- Use `tokio::runtime::Builder::new_multi_thread().enable_all().build()` for the runtime
- The rmcp `ServiceExt::serve()` handles the full 3-step handshake automatically
- `TokioChildProcess` handles newline-delimited JSON framing (NOT Content-Length)
- Do NOT create a separate reader thread — rmcp handles this internally via tokio tasks

Update `crates/tl-mcp/src/lib.rs` to add `pub mod client;` and re-export McpClient.

<task type="auto">
  <name>Task 2: Add client operations — list_tools, call_tool, ping, disconnect, server_info</name>
  <files>crates/tl-mcp/src/client.rs</files>
  <action>
Add blocking operation methods to McpClient:

1. **list_tools()** → `Result<Vec<Tool>, McpError>`:
   ```rust
   pub fn list_tools(&self) -> Result<Vec<rmcp::model::Tool>, McpError> {
       self.runtime.block_on(async {
           self.service.list_all_tools().await
               .map_err(|e| McpError::ProtocolError(e.to_string()))
       })
   }
   ```
   Uses `list_all_tools()` which handles pagination automatically.

2. **call_tool()** → `Result<CallToolResult, McpError>`:
   ```rust
   pub fn call_tool(&self, name: &str, arguments: serde_json::Value) -> Result<CallToolResult, McpError>
   ```
   - Convert arguments to `Option<serde_json::Map>` (rmcp expects Map, not arbitrary Value)
   - Create `CallToolRequestParams { name, arguments }`
   - Call `self.service.call_tool(params).await`
   - Check `result.is_error` flag — if true, return `McpError::ToolError` with content text
   - Return result

3. **ping()** → `Result<(), McpError>`:
   ```rust
   pub fn ping(&self) -> Result<(), McpError> {
       self.runtime.block_on(async {
           self.service.send_request(ClientRequest::Ping(PingRequest {})).await
               .map_err(|e| McpError::ProtocolError(e.to_string()))?;
           Ok(())
       })
   }
   ```

4. **server_info()** → `Option<&ServerInfo>`:
   Return cached server info from handshake.

5. **disconnect()** → `Result<(), McpError>`:
   ```rust
   pub fn disconnect(&mut self) -> Result<(), McpError> {
       self.runtime.block_on(async {
           self.service.cancel().await;
           Ok(())
       })
   }
   ```

6. **is_connected()** → `bool`:
   Return `!self.service.is_closed()`

Add unit tests:

- `test_client_handler_info` — verify TlClientHandler returns correct client info
